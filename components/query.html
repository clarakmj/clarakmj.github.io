insightUBC

Category: Academic project

About: I created a system that stores data about UBC courses and buildings allows users to query this data. It supports a Domain Specific Language to query the data stored. It was made using TypeScript and JavaScript with Node.js, Yarn, Mocha, Chai, Git, and IntelliJ.

The first checkpoint consisted of creating a comprehensive black box testing suite that exhausted the specification for the second checkpoint. It introduced promise chaining in order to build complex unit tests.

The second checkpoint involved building the backend to reply to the query about the courses dataset. The query was based on a given EBNF. Using asynchrony and Promise calls, I processed raw datasets by parsing, validating, and modelling the data using a data structure of my choice. This involved using JSZip.

The third checkpoint further developed the backend from the previous checkpoint to include datasets about buildings. I used parse5 to parse through HTML files into a more convenient JSON format. I also learned how to find geolocations of each building by sending a http GET request using the URL-encoded versions of the buildings' addresses.

The final checkpoint was building the frontend and connecting it altogether. I learned about RESTful API, asynchronous JavaScript, and XML (AJAX) UI testing. I built the frontend UI that allowed for building and sending queries from the UI to the Web server. Information from the UI was extracted using the browser-native global document object. An Ajax/REST request is sent where I learned to use browser-native XMLHttpRequest.
