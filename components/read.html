How Long to Read

Category: Hackathon project

About: I'm an avid reader, and my hackathon group and I had an interesting idea of how long it takes for the average reader to read a given book. It started as a curiosity that kickstarted our hackathon project.

I, alongside my group, used React.js , HTML, and CSS to create this web app that allows users to search up books to see the average length of time it takes to read a book, based on the median of user inputs. If a book does not currently exist, users also have the option of adding it to the database. From this experience, I learned how to work with React Components and props and how to implement event features.
